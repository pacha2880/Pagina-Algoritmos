# Página de Algoritmos – Prototipo

[¡VISITA LA PÁGINA AHORA!](https://pacha2880.github.io/Pagina-Algoritmos/)

Proyecto desarrollado para la materia **Interacción Humano-Computador** de la **Carrera de Informática** en la **Universidad Mayor de San Simón** (Gestión 1/2019).  
Este prototipo presenta una página en español que enseña algoritmos de forma visual y accesible.
