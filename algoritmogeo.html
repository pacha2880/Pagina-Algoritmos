Producto cruz

Definición

Supongamos que tiene tres vectores a, byc en el espacio 3D unidos en un paralelepípedo como en la siguiente imagen:

¿Cómo calcularías su volumen? Desde la escuela sabemos que debemos multiplicar el área de la base con la altura, que es la proyección de una dirección ortogonal hacia la base. Eso significa que si definimos b × c como el vector que es ortogonal a b y c y cuya longitud es igual al área del paralelogramo formado por b y c entonces | a⋅ (b × c) | Será igual al volumen del paralelepípedo. En cuanto a la integridad, diremos que b × c siempre se dirigirá de tal manera que la rotación del vector b al vector c desde el punto de b × c sea siempre en sentido contrario a las agujas del reloj (consulte la imagen a continuación).

Esto define el producto cruzado (o vector) b × c de los vectores b y c y el producto triple a⋅ (b × c) de los vectores a, by c.

Algunas propiedades notables de producto cruz:

  • a × b = − b × a
  • ( α ⋅ a ) × b = α ⋅ ( a × b )
  • Para cualquier b y c hay exactamente un vector r tal que a ⋅ ( b × c ) = a ⋅ r para cualquier vector a. De hecho, si hay dos de estos vectores r1 y r2, entonces a ⋅ ( r 1 − r 2 ) = 0 para todos los vectores a, lo que es posible solo cuando r 1 = r 2 .
  • a ⋅ ( b × c ) = b ⋅ ( c × a ) = − a ⋅ ( c × b )
  • ( a + b ) × c = a × c + b × c . De hecho, para todos los vectores r la cadena de ecuaciones es:
    r ⋅ ( ( a + b ) × c ) = ( a + b ) ⋅ ( c × r ) = a ⋅ ( c × r ) + b ⋅ ( c × r ) = r ⋅ ( a × c ) + r ⋅ ( b × c ) = r ⋅ ( a × c + b × c ) Lo cual demuestra ( a + b ) × c = a × c + b × c por el tercer punto.
  • | a × b | = | a | ⋅ | b | sin ⁡ θ donde θ es el angulo entre a y b, ya que | a × b | es igual al area del paralelogramo formado por a y b.

Dado todo esto y que la siguiente ecuación es válida para los vectores unitarios

e x × e x = e y × e y = e z × e z = 0 , e x × e y = e z ,   e y × e z = e x ,   e z × e x = e y

podemos calcular el producto cruzado de a = (x1; y1; z1) y b = (x2; y2; z2) en forma de coordenadas:

a × b = ( x 1 ⋅ e x + y 1 ⋅ e y + z 1 ⋅ e z ) × ( x 2 ⋅ e x + y 2 ⋅ e y + z 2 ⋅ e z ) = ( y 1 z 2 − z 1 y 2 ) e x + ( z 1 x 2 − x 1 z 2 ) e y + ( x 1 y 2 − y 1 x 2 )

Lo cual puede ser escrito de una forma mas elegante:

a × b = | e x e y e z x 1 y 1 z 1 x 2 y 2 z 2 | ,   a ⋅ ( b × c ) = | x 1 y 1 z 1 x 2 y 2 z 2 x 3 y 3 z 3 |

Donde | ⋅ | representa la determinante de la matriz.

En el caso 2D también se puede implementar algún tipo de producto cruzado (a saber, el producto pseudoescalar). Si quisiéramos calcular el área del paralelogramo formado por los vectores a y b, calcularíamos | e z ⋅ ( a × b ) | = | x 1 y 2 − y 1 x 2 | . Otra forma de obtener el mismo resultado es multiplicar | a | (base del paralelogramo) con la altura, que es la proyección del vector b sobre el vector a girado por 90 ∘ lo cual es a ^ = ( − y 1 ; x 1 ) . Lo que seria calcular | a ^ ⋅ b | = | x 1 y 2 − y 1 x 2 |

Si tomamos el signo en consideración, el área será positiva si la rotación de a a b (es decir, desde la vista del punto de e z ) se realiza en sentido contrario a las agujas del reloj y de lo contrario negativo. Eso define el producto pseudoescalar. Tenga en cuenta que también es igual a | a | ⋅ | b | sin ⁡ θ donde θ es el angulo entre a y b en sentido anti-horario (negativo si la rotacion en en sentido horario).

Vamos a implementar todo esto:

point3d cross(point3d a, point3d b) {
    return point3d(a.y * b.z - a.z * b.y,
                    a.z * b.x - a.x * b.z,
                    a.x * b.y - a.y * b.x);
}
ftype triple(point3d a, point3d b, point3d c) {
    return dot(a, cross(b, c));
}
ftype cross(point2d a, point2d b) {
    return a.x * b.y - a.y * b.x;
}

Propiedades

En cuanto al producto cruzado, es igual al vector cero si los vectores a y b son colineales (forman una línea común, es decir, son paralelos). Lo mismo ocurre con el producto triple, es igual a cero si los vectores a, b y c son coplanares (forman un plano común).

De esto podemos obtener ecuaciones universales definiendo líneas y planos. Una línea se puede definir a través de su dirección d vector y un punto inicial r0 o por dos puntos a y b. Se define como ( r − r 0 ) × d = 0 . En cuanto a los planos, se puede definir por tres puntos a, byc como ( r − a ) ⋅ ( ( b − a ) × ( c − a ) ) = 0 o por el punto inicial r0 y dos vectores de dirección que se encuentran en este plano d1 y ( r − r 0 ) ⋅ ( d 1 × d 2 ) = 0 .

En 2D, el producto pseudoescalar también se puede usar para verificar la orientación entre dos vectores porque es positivo si la rotación del primer al segundo vector es en el sentido de las agujas del reloj y, de lo contrario, es negativa. Y, por supuesto, se puede usar para calcular áreas de polígonos, que se describe en un artículo diferente. Un producto triple se puede utilizar para el mismo propósito en el espacio 3D.
