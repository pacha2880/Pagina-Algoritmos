Cola

Definición

Una cola es una estructura de datos lineal FIFO, First In First Out. El primer elemento en entrar a la estructura sera el primero en salir.

Su funcionamiento similar al de una fila/cola de, por ejemplo, un banco.

Declaracion y operaciones

Declaracion: queue<tipo_de_dato> nombre

Tiene las siguientes operaciones:

  • empty
  • size
  • front
  • back
  • push
  • pop

empty devuelve un valor de tipo booleano indicando si la cola esta vacia

size devuelve un valor de tipo entero indicando la cantidad de elementos dentro de la cola

front devuelve el valor del frente de la cola

back devuelve el valor del final de la cola

push añade un elemento al final de la cola

pop elimina el elemento del frente de la cola

Ejemplo

queue<int> cola; // Crea una cola
cola.push(5);
cola.push(2);
cola.push(4);
// La cola tiene los elementos 5, 2, 4 del frente al final

cola.pop(); // Elimina el elemento del frente
cout << cola.size() // Imprime 3, el tamaño de la cola
cout << cola.front() // Imprime 2, el elemento del frente

Cuestionario

¿Si se tienen las siguientes operaciones: cola.push(2); cola.push(3); cola.push(1); cola.pop(); que valor devolveria cola.front()?

¿Que pasa si haces cola.pop() a una cola vacia, que pasa?
