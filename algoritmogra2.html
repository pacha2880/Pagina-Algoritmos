Puntos de articulacion

Definición

Nos dan un gráfico no dirigido. Un punto de articulación (o vértice cortado) se define como un vértice que, cuando se elimina junto con los bordes asociados, desconecta el gráfico (o, más precisamente, aumenta el número de componentes conectados en el gráfico). La tarea es encontrar todos los puntos de articulación en el gráfico dado.

El algoritmo descrito aquí se basa en la primera búsqueda en profundidad y tiene complejidad O (N + M), donde N es el número de vértices y M es el número de bordes en la gráfica.

Algoritmo

Elija un vértice arbitrario de la raíz del gráfico y ejecute la búsqueda en profundidad desde él. Tenga en cuenta el siguiente hecho (que es fácil de probar):

  • Digamos que estamos en el DFS, mirando a través de los bordes comenzando desde vértice v ≠ raíz. Si el borde actual (v, a) es tal que ninguno de los vértices ao sus descendientes en el árbol transversal DFS tiene un borde posterior a ninguno de los antepasados de v, entonces v es un punto de articulación. De lo contrario, v no es un punto de articulación.
  • Consideremos el caso restante de v = root. Este vértice será el punto de articulación si y solo si este vértice tiene más de un hijo en el árbol DFS.

Ahora tenemos que aprender a verificar este hecho para cada vértice de manera eficiente. Usaremos el "tiempo de entrada en el nodo" calculado por la primera búsqueda en profundidad.

Por lo tanto, digamos que tin [v] denota el tiempo de entrada para el nodo v. Introducimos una matriz baja [v] que nos permitirá verificar el hecho de que cada vértice v. Baja [v] es el mínimo de tin [v], los tiempos de entrada tin [p] para cada nodo p que está conectado al nodo v a través de un borde posterior (v, p) y los valores de low [to] para cada vértice del cual es un descendiente directo de v en el árbol DFS:

l o w [ v ] = min { t i n [ v ] t i n [ p ]  for all  p  for which  ( v , p )  is a back edge l o w [ t o ]  for all  t o  for which  ( v , t o )  is a tree edge

Ahora, hay un borde posterior del vértice v o uno de sus descendientes a uno de sus antepasados si y solo si el vértice v tiene un hijo para el cual bajo [to] <tin [v]. Si baja [to] = tin [v], el borde posterior llega directamente a v; de lo contrario, se trata de uno de los antepasados de v.

Por lo tanto, el vértice v en el árbol DFS es un punto de articulación si y solo si l o w [ t o ] ≥ t i n [ v ]

Implementacion

La implementación debe distinguir tres casos: cuando bajamos el borde en el árbol DFS, cuando encontramos un borde posterior a un antepasado del vértice y cuando regresamos a un padre del vértice. Estos son los casos:

  • v i s i t e d [ t o ] = f a l s e - la arista es parte del arbol DFS
  • v i s i t e d [ t o ] = t r u e - la arista es un back edge (se conecta a uno de sus ancestros que no es su padre)
  • t o = p a r e n t - la arista se conecta al padre del arbol DFS

Para implementar esto, debemos hacer una funcion DFS que reciba como parametro el padre del nodo actual.

Implementacion en C++:

int n; // number of nodes
vector> adj; // adjacency list of graph

vector visited;
vector tin, low;
int timer;

void dfs(int v, int p = -1) {
    visited[v] = true;
    tin[v] = low[v] = timer++;
    int children=0;
    for (int to : adj[v]) {
        if (to == p) continue;
        if (visited[to]) {
            low[v] = min(low[v], tin[to]);
        } else {
            dfs(to, v);
            low[v] = min(low[v], low[to]);
            if (low[to] >= tin[v] && p!=-1)
                IS_CUTPOINT(v);
            ++children;
        }
    }
    if(p == -1 && children > 1)
        IS_CUTPOINT(v);
}

void find_cutpoints() {
    timer = 0;
    visited.assign(n, false);
    tin.assign(n, -1);
    low.assign(n, -1);
    for (int i = 0; i < n; ++i) {
        if (!visited[i])
            dfs (i);
    }
}

La función principal es find_cutpoints; realiza la inicialización necesaria y comienza la búsqueda en profundidad en cada componente conectado del gráfico.

La función IS_CUTPOINT (a) es una función que procesará el hecho de que el vértice a es un punto de articulación, por ejemplo, imprimirlo (Tenga en cuenta que esto se puede llamar varias veces para un vértice).
